# Public Scope

This repository contains generalized RDTC v1 algorithm, RTL, verification, FPGA integration, and implementation assets.

It does not include product-specific protocols, non-public data, third-party generated IP, commercial technology libraries, PDK payloads, implementation databases, or board projects. External FPGA IP must be generated locally from the documented interface and version requirements.

Only the published RDTC v1 lossless codec scope is represented. PointCloud RTL, RLE_RICE RTL, AXI-MM/DMA descriptor integration, and physical-signoff results are not included or claimed.
