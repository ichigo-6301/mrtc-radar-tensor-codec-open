# Overview

RDTC v1 is a streaming lossless-compression IP within the MRTC framework for block-organized complex Range-Doppler samples. This release covers RAW_BYPASS, ZERO_RICE, DELTA_RICE, stream interfaces, control/status, and the published verification assets only.

The IP connects radar-sensing data producers to storage, transport, or downstream processing. It does not define upstream radar processing and does not include PointCloud, DMA descriptors, or AXI-MM system integration.
