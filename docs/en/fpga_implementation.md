# FPGA Implementation

The FPGA implementation / emulation target checks RDTC RTL synthesizability, AXI backpressure behavior, wrapper connectivity, and implementation timing conditions. This repository provides handwritten RTL, a constraint template, and integration guidance. Board clocks, IO, DDR, external IP, and generated projects are platform-owned inputs.

This release does not claim a board PASS. Any FPGA frequency or resource result must identify the device, tool version, constraints, and implementation report.
